# image_download
一个通过lolicon的api进行大量爬取图片的工具  
本代码使用python编写，属于是纯个人练手项目，不过你仍然可以使用  
当然，如果你不嫌弃这代码的炸裂程度  

## 使用方法
运行脚本：
```bash
python _image_please_25_07_30.py
```
图片将会保存在同目录的文件夹images处

## 依赖
```bash
pip install requests lxml
```
